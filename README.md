🚀 Cara Menjalankan Secara Lokal
Clone repository ini:

Bash
git clone [https://github.com/Irsyad-ald/portofolio-main.git](https://github.com/Irsyad-ald/portofolio-main.git)
cd portofolio-main
Install dependencies:

Bash
npm install
    Jalankan server lokal:

Bash
npm run dev
    Build untuk production:

Bash
npm run build
